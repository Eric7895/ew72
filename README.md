# Data Science Portfolio

A personal portfolio website for presenting my background, technical interests, and selected data science projects.

## Built With

- React
- Vite
- React Router
- JavaScript
- CSS

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Project Structure

- `src/components` — reusable components
- `src/pages` — homepage and project-detail pages
- `src/data/projects.js` — project content
- `src/assets` — images and other media
- `src/App.jsx` — application routes
- `src/App.css` and `src/index.css` — styling
